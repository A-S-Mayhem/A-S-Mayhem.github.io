## Course Description
This course covers the analysis and design of robotic systems, with a focus on:

- Understanding and analysis of robotics system espically in Aerial Robots.
- Software for robot simulation and deployment.

## Student Learning Outcomes

- Understand and apply kinematics and dynamics principles application to robotics
- Understand engineering applications of robotic systems
- Using software tools (MATLAB, Solidworks) for solving robotic systems
