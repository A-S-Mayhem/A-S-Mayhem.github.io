[Github-Repo](https://github.com/antonioalt23/The_Code_Blinders) : The Code Blinders is a team of programmers that tried to democratize the knowledge of programming in python for students of Public Universities in Peru. In this bootcamp, we taught basic programming concepts such as data structures, defining functions, and working with the essential specialized libraries, especially Numpy, Pandas, Google APIs and Selenium.
