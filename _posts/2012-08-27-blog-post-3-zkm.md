可以将多个条目组织成有序或无序列表。  

有序列表
======
要创建有序列表，请在每个列表项前添加数字并紧跟一个英文句点。数字不必按数学顺序排列，但是列表应当以数字`1. `起始。  
```
做一名优秀研究生，必须具备批判性的思维
1. 正确分析负面结果（negative results）：负面磨炼
2. 耗费大量时间的完美主义阻碍创新进取:保持往前走
3. 科研文献与学术讲座的取与舍:权衡时间分配
4. 挑战传统思维：Follow the logic
```
渲染效果如下：  

做一名优秀研究生，必须具备批判性的思维
1. 正确分析负面结果（negative results）：负面磨炼
2. 耗费大量时间的完美主义阻碍创新进取：保持往前走
2. 科研文献与学术讲座的取与舍：权衡时间分配
2. 挑战传统思维：Follow the logic

#无序列表
要创建无序列表，请在每个列表项前面添加波折号、星号或加号。缩进一个或多个列表项可创建嵌套列表。  
```
做一名优秀研究生，必须具备批判性的思维
- 正确分析负面结果（negative results）：负面磨炼
- 耗费大量时间的完美主义阻碍创新进取:保持往前走
- 科研文献与学术讲座的取与舍:权衡时间分配
- 挑战传统思维：Follow the logic
```

做一名优秀研究生，必须具备批判性的思维
