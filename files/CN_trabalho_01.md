# I Trabalho de Cálculo Numérico 

## Grupo XX

Estudar o Método da Falsa Posição. Implementar um algoritmo para analisar o seu desempenho computacional. Comparar com o Método da Bissecção e com o Método de Newton.

### Comparação entre os métodos
=====

Para comparar os métodos acima descritos, o grupo deve definir um problema modelo para fazer o estudo. Para  esse problema, implementar os três métodos e comparar, para um mesmo chute, número de iteradas, o tempo de execução e/ou outra forma que o grupo encontre de evidenciar as principais características de cada método, em particular o método em estudo.
