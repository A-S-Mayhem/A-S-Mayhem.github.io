## Para os grupos 01, 02 e 03

Estudar o método atribuído a cada grupo. Implementar um algoritmo para analisar o seu desempenho computacional. Comparar com o Método da Bissecção e com o Método de Newton na resolução de uma equação do tipo $f(x) = 0$.

### Comparação entre os métodos

Para comparar os métodos acima descritos, o grupo deve definir um problema modelo para fazer o estudo. Para  esse problema, implementar os três métodos e comparar, para um mesmo chute, número de iteradas, o tempo de execução e/ou outra forma que o grupo encontre de evidenciar as principais características de cada método, em particular o método em estudo. Fazer vários exemplos e execute cada algoritmo com vários chutes.

Cada grupo deve encaminhar um texto, em formato pdf, com os experimentos (gráficos, tabelas, outros) e com a análise dos resultados, comentando o que fora observado em cada experimento.

#### Tema para cada Grupo

* Grupo 01 - Método da Falsa Posição
* Grupo 02 - Método da Secante
* Grupo 03 - Método Quociente-Diferença (Para funções polinomiais)

## Para o grupo 04

Implementar o Método de Newton para determinar o(s) ponto(s) crítico(s) de uma função $f:\mathbb{R}^n \to \mathbb{R}$.

Fazer vários exemplos. Para cada exemplo, apresentar comentários, tabelas e gráficos evidenciando as características observadas. Faça testes com chutes próximos e distantes do ponto crítico. Fazer um gráfico mostrando as curvas de nível de $f:\mathbb{R}^2 \to \mathbb{R}$ e os pontos $x_k \in \mathbb{R}^2$ se aproximando do ponto crítico $x^* \in \mathbb{R}^2$.

## Para o grupo 05
Implementar o Método de Newton para resolver um sistema de equações não lineares.  Para cada exemplo, apresentar comentários, tabelas e gráficos evidenciando as características observadas. Testar várias funções com vários chutes iniciais.
