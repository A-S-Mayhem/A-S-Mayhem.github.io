# I Trabalho de Cálculo Numérico 


Estudar o método atribuído a cada grupo. Implementar um algoritmo para analisar o seu desempenho computacional. Comparar com o Método da Bissecção e com o Método de Newton.

## Comparação entre os métodos

Para comparar os métodos acima descritos, o grupo deve definir um problema modelo para fazer o estudo. Para  esse problema, implementar os três métodos e comparar, para um mesmo chute, número de iteradas, o tempo de execução e/ou outra forma que o grupo encontre de evidenciar as principais características de cada método, em particular o método em estudo. Fazer vários exemplos e execute cada algoritmo com vários chutes.

Cada grupo deve encaminhar um texto, em formato pdf, com os experimentos (gráficos, tabelas, outros) e com a análise dos resultados, comentando o que fora observado em cada experimento.

### Tema para cada Grupo

* Grupo XX - Método da Falsa Posição
* Grupo XX - Método Quociente-Diferença
