This project serves as a refresher for Object Orient Programming concepts - Abstraction, Encapsulation, Inheritance and Polymorphism.
In addition, a practice project implementing multi level tic-tac-toe board game in Python. Currently, the game can be played in the python console. Future versions may be deployed on Streamlit or any other GUI platform.
