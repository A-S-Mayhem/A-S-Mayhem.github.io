## What it Does
The Carbon Footprint Tracker allows users to log daily activities, such as biking, eating salads, or gardening, and receive points based on the environmental impact of these actions. Users can visualize their progress through dynamic line and pie charts, track their goal achievements, and advance through an engaging ranking system inspired by nature-themed icons.

## How We Built It
- **Technologies Used:** HTML, CSS, and JavaScript.
- **Data Visualization:** Integrated the Chart.js library to create responsive and visually appealing charts.
- **Data Persistence:** Leveraged LocalStorage to enable activity logging and progress tracking across sessions.
- **User Interface:** Styled for accessibility and simplicity to ensure ease of use.

## Challenges We Ran Into
- Designing a seamless user experience while balancing functionality with simplicity.
- Implementing dynamic updates for the visualizations to reflect real-time activity logging.
- Calibrating the ranking system to provide meaningful progression while maintaining user motivation.

## Accomplishments That We're Proud Of
- Successfully creating an interactive platform that encourages eco-friendly habits.
- Developing visually appealing and responsive charts to dynamically represent user data.
- Designing an intuitive ranking system that rewards consistent effort and promotes user engagement.

## What We Learned
- Enhanced our understanding of data visualization techniques using Chart.js.
- Refined our JavaScript skills for handling real-time updates.
- Gained experience in leveraging LocalStorage for data persistence.
- Learned how to design engaging and user-friendly interfaces.

## What's Next for Carbon Footprint Tracker
- Expanding activity categories to include more sustainable practices.
- Introducing personalized recommendations based on user activity trends.
- Incorporating social features to allow users to share their progress and compete with friends.
- Developing a mobile app version for greater accessibility and convenience.
